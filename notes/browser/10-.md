``` javascript
///////////////////////////////////////////////////////////////////////////////////////////////////////////////////////


// 浏览器输入 url 后会 发生什么

1. 浏览器补全 url 地址( 对地址中非 ASCII 字符会自动完成 url 编码 )

2. 根据请求报文发起请求
    a. 请求行
    b. 请求头
    c. 请求体

3. 浏览器接受到响应, 并根据响应头中的 Content-Type 解析 响应体( 响应成功并返回 text/html )
    a. 从上到下执行
    b. 解析响应
        - 发现 link 标签，将相对路径转换为完整的 url 地址( 相对路径相对于 网站的 根目录 )
        - 发现 img 标签( src 不会阻塞 浏览器解析 )，有且仅当有了响应结果，则会吧图片渲染到页面上
        - 发现 a 标签，不发送请求，显示到页面上，仅在点击时解析
        - 发现 script 标签( 阻塞解析 )，请求该资源完成后，执行并继续解析
        ...


///////////////////////////////////////////////////////////////////////////////////////////////////////////////////////
```
