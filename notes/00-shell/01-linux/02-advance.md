#

#### tail

``` javascript
///////////////////////////////////////////////////////////////////////////////////////////////////////////////////////


// 通常用于监控日期


tail -[show_line_number][f/F] [file_path]          // 实时监控文件

    -f: 追踪文件名( --follow=descriptor )            // 根据文件描述符进行追踪，当文件改名或被删除，追踪停止

    -F: 追踪文件索引( --follow=name --retry )        // 根据文件名进行追踪，并保持重试，即该文件被删除或改名后
                                                   // 如果再次创建相同的文件名，会继续追踪


///////////////////////////////////////////////////////////////////////////////////////////////////////////////////////
```
