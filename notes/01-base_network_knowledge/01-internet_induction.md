#### 冯诺伊曼式计算机五大组成部分

``` javascript
///////////////////////////////////////////////////////////////////////////////////////////////////////////////////////


/**
 * 冯诺伊曼式计算机五大组成部分
 * 
 * 冯诺伊曼( 计算机之父 )
 */


    运算器
    
        CPU: 整数，逻辑运算
        
        GPU: 小数，精确运算
    
    存储器
    
        内存: 断电数据清空，读写速度快
        
        硬盘: 辅存，数据可持久化，读写速度相对较慢
    
    控制器
    
        主板上的一些器件
    
    输入设备
    
        键盘，鼠标，麦克风 ...     网口 ------- +   
                                             ｜
    输出设备                                   +   即是输入设备，也是输出设备
                                             ｜
        显示器，耳机 ...          网口 ------- +


///////////////////////////////////////////////////////////////////////////////////////////////////////////////////////
```
