# sentos

#### directory introduction

``` javascript
///////////////////////////////////////////////////////////////////////////////////////////////////////////////////////


.                                               // 根目录: 仅存放文件夹( 不存放文件 )
|
├── /bin -> usr/bin                             // 可执行二进制文件目录
├── /boot                                       // 放置linux系统启动文件
├── /dev                                        // 存放linux系统下的设备文件( eg: 挂载光驱 )
├── /etc                                        // 系统配置文件
├── /home                                       // 系统默认用目录文件( eg: 新建的用户会在此目录下新建 )
├── /lib -> usr/lib                             // 系统使用函数库的目录
├── /lib64 -> usr/lib64                         //
├── /lost+found                                 // 系统异常产生错误时, 会将部分遗失的片段放置于此
├── /mnt:/media                                 // 光盘默认挂载点
├── /mnt                                        // 
├── /opt                                        // 用于给主机阿安装额外软件的目录
├── /proc                                       // 该目录下都存储于内存中( eg: 系统核心, 外部设备, 网络状态... )
├── /QL                                         // 
├── /root                                       // 系统第一个启动的分区, 管理员root的目录
├── /sbin & /usr/sbin & /usr/loocal/sbin        // readyonly, 存放系统管理员root的可执行命令
├── /run                                        // 
├── /srv                                        // 服务启动后需要访问的数据目录( eg: www服务需要的src/www )
├── /sys                                        //
├── /tmp                                        // 存放用户执行程序的临时文件
├── /usr                                        // 
└── /var                                        //


///////////////////////////////////////////////////////////////////////////////////////////////////////////////////////
```
