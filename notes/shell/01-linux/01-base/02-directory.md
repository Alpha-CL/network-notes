``` javascript
        |                                         + ---> + -------------> +      |      |

        + ---------------------------------------------- +                       + ---- +


//-------------------------------------------------------------------------------------------------------------------//


|--- 存放命令
|
|
|- bin -> usr/bin           // 存放命令命令
|- /sbin                    // 保存与系统环境设置相关的命令
|                           // 只有 root 可以使用这些命令进行系统环境设置( 也有些命令可以允许普通用户查看 ) 


//- - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - -//


|--- 用户文件目录( 普通用户 | root )
|
|
|- home                     // 普通用户的主目录
|- root                     // root 用户目录
|


//- - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - -//


|--- 系统文件目录
|
|
|- tmp                      // 临时目录。系统存放临时文件的目录，在该目录下
|                           // 所有用户都可以访问和写入( 建议此目录中不能保存重要数据，最好每次开机都把该目录清空 )
|
|- srv                      // service: 服务数据目录, 一些系统服务启动之后，可以在这个目录中保存所需要的数据
|
|- etc                      // etcetera: 存放所有的系统管理所需要的配置文件和子目录
|
|- lib                      // library: 系统最基本的动态连接共享库
|


|--- 虚拟文件
|
|
|- proc                     // processes: 虚拟文件系统
|                           //            存储的是当前内核运行状态的一系列特殊文件
|
|- sys                      // systetm: 虚拟文件系统,
|                           //          该目录中的数据都保存在内存中( 主要保存与内核相关的信息 )
|

|--- 系统启动目录
|
|
|- boot                     // 启动 linux的时 使用的 核心文件( 包括一些 连接文件 及 镜像文件 )
|
|- run --> /var/run         // 临时文件: 存储系统启动后的信息
|                           //         ( 重启后重新生成对应的目录数据 )
|


//- - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - -//


|- usr                      // unix source: 软件安装包
|   |
|   |
|   |- bin                  // 
|   |- sbin                 //
|   |- lib                  //
|   |- XIIR6                //
|   |- local                //
|   |- share                //
|   |- src                  //
|   |- include              //
|


|- var                      // variable: 存储动态数据，例如缓存、日志文件、软件运行过程中产生的文件等
|   |
|   |
|   |- lib                  // 
|   |- log                  // 登陆文件放置的目录
|   |- run                  // 存放服务运行后 PID
|   |- spool                // 临时存放, 随时会被用户所调用的数据
|   |- www                  // RPM 包安装的 Apache 的网页主目录
|   |- nis/yp               // 
|   |- tmp                  // 一些应用程序在安装或执行时, 需要在重启后使用的某些文件, 此目录能将该类文件暂时存放起来, 完成后再行删除
|   |- ...
|


|
|- opt                      // option: 默认为空, 存放各种安装包
|
|
|


//- - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - -//


|--- 外部设备挂载的目录
|
|- media                    // 软盘/光盘等 挂在目录
|- mnt                      // U盘/硬盘等 挂载目录
|- misc                     // 自定义挂载目录
|- dev                      // device: 存放的是 Linux 的外部设备
|


//- - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - -//


|--- 其他
|
|- lost + found             // 一般情况下是空的，当系统非法关机后，这里就存放了一些文件
|- selinux                  // Redhat/CentOS 特有文件: 安全机制的防火墙文件
|


//- - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - -//


// file system formatting

    - 深蓝是文件夹/浅蓝是快捷方式( 软链接 )

    - 


///////////////////////////////////////////////////////////////////////////////////////////////////////////////////////
```
